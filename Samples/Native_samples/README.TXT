INTRODUCTION:
	This samples has been implemented using Java Native Interface. 
	Every example has a descriptive title and with number. 

Folders: 

	modules: Library modules source (we call to library functions using jni)
	cmake_tools: utils for cmake scripting code.

Prerequisites: 
	Good knowledge in c and c++.
	Java and Android basic knowledge
	Cmake knowledge.

Install programs: 
	Cmake. 
	C++ compiler. 
	Android sdk.
	Android ndk.
	JDK installed.
	
INSTALLATION: 

open cmake-gui: 
select your operative system. 
If you are using android select your api and put where is your ndk.
